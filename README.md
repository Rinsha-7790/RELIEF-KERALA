# 🌊 Disaster Relief Coordination System – Kerala

A web-based Disaster Relief Coordination System developed using Django to connect donors, volunteers, relief camps, panchayats, and administrators during disasters such as floods and landslides.

---## ✨ Features

- 🏠 User-friendly home page
- 👤 Secure user registration and login
- 🤝 Volunteer registration and management
- 🏕️ Relief camp creation and management
- 📦 Item donation management
- 💰 Online monetary donations
- 📍 Disaster location tracking
- 📢 Emergency announcements and notifications
- 📊 Admin dashboard with statistics
- 🛡️ Role-based access control (Admin, Staff, Volunteer, Donor)
- 📄 ID proof verification for volunteers
- 🔍 Transparent donation tracking

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Rinsha-7790/RELIEF-KERALA.git
```

2. Go to the project folder

```bash
cd RELIEF-KERALA
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

5. Install the required packages

```bash
pip install -r requirements.txt
```

6. Apply database migrations

```bash
python manage.py migrate
```

7. Run the development server

```bash
python manage.py runserver
```

8. Open your browser

```
http://127.0.0.1:8000/
```
